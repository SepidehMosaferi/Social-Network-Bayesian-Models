# Social-Network-Bayesian-Models

This repository contains all R code for the paper "**Bayesian Modeling of Social Networks with Covariate-dependent Error Rates**" by Sepideh Mosaferi, Shonosuke Sugasawa, and James Kitts.

The repository is structured as follows:

* The folder “Application” contains all the code to run the paper’s applications.

* The folder “Data” contains all the datasets used in the paper.

* The folder “Simulation” contains all the code to run the paper’s simulations.
